# **Atividade Coding Mobile**

## Utilização de API com Responsividade

Elaboração de plataforma web com busca para alguma API pública. Foi utilizada a api do ITunes.

## **O que ela entrega**

O site foi desenvolvido com media queries para se adaptar a diferentes tamanhos de tela.
O projeto foi pensado com prioridade para visualização em dispositivos móveis, conforme solicitado na atividade.
Atribuido ao projeto o botão de favoritos, limpeza de cache e artistas mais ouvidos.

💻 Desktop: layout com imagem lateral
📱 Tablet: ajuste de espaçamento e fontes
📲 Mobile: layout vertical priorizando a leitura e interação

A plataforma faz uma busca rápida do artista, música ou banda e nos devolve 20 cards com o resultado da psesquisa, podendo ser ouvido por 30 segundos 

## **Linguagens Utilizadas**

HTML para maracação das funções exibidas na página
CSS para estilização, utilizada uma paleta discreta com um botão que chame atenção para realização da busca
Java Script para configuração da API 

## **API Utilizada**

Os dados das músicas são obtidos através da API pública do iTunes, que fornece informações sobre artistas, músicas e álbuns.

Endpoint utilizado: [https://itunes.apple.com/search?term=]
A aplicação realiza uma requisição que por fim recebe os dados solicitados pelo buscador.
